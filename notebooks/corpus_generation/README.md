# Corpus-generation notebook 
This folder contains the original notebook used for TamilSpell corpus construction. 

## File 
| File | Description | 
|---|---| 
| `tamilspell_corpus_generation_original.ipynb` | Original notebook containing the corpus-construction workflow | 

## What the notebook contains 

The notebook includes code for: 
1. defining Tamil character lists,
2. generating UyirMei character mappings,
3. creating keyboard-neighbour dictionaries,
4. generating non-word errors,
5. generating isolated real-word, non-word, and sandhi errors,
6. generating 2-way error combinations, and
7. generating 3-way error combinations.

## Notes 
This notebook is preserved as the original construction workflow. It may contain code that was later reorganised or cleaned for reproducibility.
