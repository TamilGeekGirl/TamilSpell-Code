# Notebooks

This directory contains Jupyter notebooks associated with TamilSpell.

## Subdirectories

```text
corpus_generation/    Original notebook used for TamilSpell corpus construction
use_cases/            Notebooks demonstrating TamilSpell use cases

```

## Notes

The notebooks are provided for transparency and reproducibility.

The full TamilSpell corpus is not stored in this repository. Download the full corpus using the official DOI:

10.15132/10000260
