# Lexical input files 
This folder contains lexical input files used during TamilSpell corpus construction. 
## Files 
| File | Description | 
|---|---| 
| `all_tamil_words.txt` | Tamil word list used for generating synthetic non-word errors | 
| `correct_sentences.txt` | Correct Tamil sentences used for sentence-level error generation | 

## Use in the code 

`all_tamil_words.txt` is used for word-level non-word error generation. 
`correct_sentences.txt` is used for sentence-level non-word error generation. 

## Notes 

These files are input files only. They are not the full published TamilSpell corpus.
