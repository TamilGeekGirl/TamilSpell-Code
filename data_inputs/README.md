# Data input files 

This directory contains the input files used by the TamilSpell corpus-construction code. These files are not the full published TamilSpell corpus. 

The final corpus is available through the official DOI: ```text 10.15132/10000260

## Subdirectories
lexical/              Tamil word lists and correct sentence lists
real_word_errors/     Input files for real-word errors
sandhi_errors/        Input files for sandhi errors
combination_inputs/   Input files for 2-way and 3-way error combinations

## Notes

The files in this directory are retained for reproducibility and transparency. They are source/input files used by the notebooks, not the final published corpus.
